# C语言实验课示例代码

本仓库包含C语言实验课的示例代码，旨在帮助同学们理解和掌握C语言基础编程概念。

主要目的是提供一个用于参考的代码库，同学们可以根据自己的需要去看对应实验组的源代码。我使用的环境是MacOS+Clang,实验环境因人而异，不需要和我保持一致，但是代码的思路是通用的。

## 📂 仓库结构

```
C_Course_Code/
├── exp_1/              # 第一次实验：基础C语言练习
│   ├── exp_1_1_*.c     # 字符处理、类型判断
│   ├── exp_1_2_*.c     # 循环：九九乘法表
│   ├── ...
│   └── exp_1_10_*.c    # 用户输入：四则运算器
├── exp_2/              # 第二次实验
├── exp_.../            # 其他实验
├── hello.c             # 基础示例：Hello World
└── README.md           # 本文档
```

## 文件命名规范

```
exp_{实验号}_{题目号}_{功能描述}.c
```

例如：`exp_1_3_triangle_area.c` 表示实验一第3题：三角形面积计算
